# 重装前的准备
> 笔者使用的是华硕的笔记本电脑，仅以此为例。
- 将必要的文件进行备份
- 准备一个8G以上U盘
- 下载[Windows11官方镜像](https://www.microsoft.com/zh-cn/software-download/windows11)
- 下载笔记本厂商驱动，以华硕为例，前往[下载中心](https://www.asus.com.cn/support/download-center/)
# 装机流程
> 华硕主板需要在BIOS中[关闭VMD](https://www.asus.com.cn/support/faq/1044458/)，防止在系统无法识别磁盘控制器。注:关闭VMD技术将会导致您的电脑无法使用 RAID 磁盘阵列。
1. 使用Ventoy将U盘刷写
2. 将Windows镜像ISO和笔记本厂商驱动等文件copy到刷好的U盘内
3. 插上U盘，进行重启，期间按`F2`进入BIOS
4. 将U盘设置为第一启动项
5. 若无法进入ventoy的引导界面，请根据[官方教程](https://www.ventoy.net/cn/doc_secure.html)操作，或者直接关闭 **Secure Boot**，但会影响到反作弊程序的运行。
6. 进入Windows 11系统镜像安装界面后，重点将分区列表清理干净(建议如此)，比如原系统安装在磁盘0,建议提前将磁盘0中的重要文件转移到磁盘1或者其他存储设备中，然后在这里清空所有磁盘0的分区，再选中磁盘0进行装载系统。
7. 等待Windows 11安装系统，直到引导界面出现，这里默认需要插上网线，进行联网驱动更新，但也可以手动选择提前在U盘中准备的厂商驱动。
8. 在进入系统后，依次安装准备好的笔记本厂商提供的驱动程序，随后进行关机重启。
# Windows初始化
1. 重新进入系统后，进入 **隐私和安全性** -> **设备加密**，选择关闭。(这里重装完系统后，趁文件还没有那么多，解密很快)如果你有设备加密的需要，可以选择保持开启，但一定要在激活Windows之后，登录微软账户，因为恢复密钥只有微软官方提供给你。
2. 下载[Office Tool Plus](https://otp.landian.vip/zh-cn/)，进入程序后，遵照[官方给的快速配置教程](https://blog.yerong.org/archives/11)，点击命令按钮或者按下 `Ctrl` + `Shift` + `P` 打开命令框，输入以下命令，安装office三件套(Word、PPT、Excel)：
```
deploy /add O365ProPlusRetail_zh-cn /O365ProPlusRetail.exclapps Access,Groove,Lync,OneDrive,OneNote,Outlook,OutlookForWindows,Publisher,Teams /channel Current /dlfirst /edition 64
```
3. 虽然在 Office Tool Plus 中有使用KMS激活Office的功能，但这里还是推荐使用 [MAS](https://massgrave.dev/index.html) 来激活Windows和Office，遵照官网，打开Windows自带的Powershell终端(只需要右键任务栏的Windows图标，然后选择终端打开即可)，输入命令：`irm https://get.activated.win | iex` 如果网络通畅，即可进入交互式脚本。
4. 进入脚本后，键入`1`即可使用数字权利激活Windows，跟随提示回到首页，键入`2`安装Ohook即可激活Office。关于 **KMS**、**HWID**、**Ohook** 的相关信息，可以来到 [雨月空间站的文章](https://www.mintimate.cn/2023/12/26/whatOhook/) 了解。

# 软件推荐

- [Steamcommunity 302](https://www.dogfight360.com/blog/18682/) | 这是由羽翼城大佬制作的加速工具，可以用来直连GitHub、Steam等一些国外平台。
- [g-helper](https://g-helper.com/) | 一款用来代替华硕笔记本奥创中心的软件，主要进行CPU和GPU频率、风扇的调整，比较好用的是独显直连和三种预设切换的丝滑，可以释放出奥创达不到的性能。
- [everything](https://www.voidtools.com/zh-cn/downloads/) | 一款强大的检索工具，笔者的日常使用已经脱离不开了，能够极其快速的找到你电脑上的任意一个文件。
- [Snipaste](https://zh.snipaste.com/) | 一款简洁好用的截图贴图工具
- [notepad3](https://rizonesoft.com/downloads/notepad3/) | 一款记事本平替工具，一定程度上还能平替vscode编辑器，提供多种语法的高亮，启动快速，简洁好用。
- [imageglass](https://imageglass.org/) | 图片查看器(支持一定的图片编辑)
- [mpv_PlayKit(原mpv-lazy)](https://github.com/hooke007/mpv_PlayKit) | 由mpv爱好者打造的魔改版视频浏览器，主打懒人一键配置，启动速度快，简洁好用。
- [NanaZip](https://nanazip.com/) | 基于7Zip的现代化压缩工具，可在微软商店一键下载安装。
